# <a name="updateint-method-sqlserverresultset"></a>Método updateInt (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

|NOMBRE|Descripción|  
|----------|-----------------|  
|[updateInt (int, int)](../../../connect/jdbc/reference/updateint-method-int-int.md)|Actualiza la columna designada con un valor **int** según el índice de la columna.|  
|[updateInt (java.lang.String, int)](../../../connect/jdbc/reference/updateint-method-java-lang-string-int.md)|Actualiza la columna designada con un valor **int** según el nombre de la columna.|

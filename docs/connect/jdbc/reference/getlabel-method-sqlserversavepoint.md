# <a name="getlabel-method-sqlserversavepoint"></a>Método getLabel (SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Obtiene el nombre de la etiqueta del punto de retorno.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.lang.String getLabel()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un objeto **String** que contiene el nombre de la etiqueta del punto de retorno.

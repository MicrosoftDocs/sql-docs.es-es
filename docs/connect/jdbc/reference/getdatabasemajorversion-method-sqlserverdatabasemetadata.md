# <a name="getdatabasemajorversion-method-sqlserverdatabasemetadata"></a>Método getDatabaseMajorVersion (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el número de versión principal de la base de datos subyacente.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public int getDatabaseMajorVersion()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un valor **int** que indica la versión principal de la base de datos.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Observaciones  
 El método getDatabaseMajorVersion especifica este método getDatabaseMajorVersion en la interfaz java.sql.DatabaseMetaData.

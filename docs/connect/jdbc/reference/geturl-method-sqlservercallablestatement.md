# <a name="geturl-method-sqlservercallablestatement"></a>Método getURL (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como un objeto URL en el lenguaje de programación Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|NOMBRE|Descripción|  
|----------|-----------------|  
|[getURL (int)](../../../connect/jdbc/reference/geturl-method-int.md)|Recupera el valor del parámetro designado como un objeto URL en el lenguaje de programación Java según el índice del parámetro.|  
|[getURL (java.lang.String)](../../../connect/jdbc/reference/geturl-method-java-lang-string.md)|Recupera el valor del parámetro designado como un objeto URL en el lenguaje de programación Java según el nombre del parámetro.|

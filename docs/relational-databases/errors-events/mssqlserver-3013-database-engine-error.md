|Atributo|Value|
|---|---|
|Nombre de producto|SQL Server|
|Id. de evento|3013|
|Origen de eventos|MSSQLSERVER|
|Componente|SQLEngine|
|Nombre simbólico|DMP_ABORT|

Este mensaje indica un error general.

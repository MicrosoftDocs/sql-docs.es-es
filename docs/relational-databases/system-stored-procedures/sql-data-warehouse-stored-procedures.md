>  Algunos procedimientos almacenados del sistema adicionales solo se usan dentro de una instancia de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] o a través de las API de cliente y no están pensados para uso general de los clientes. Estos procedimientos se enumeran en [procedimientos almacenados del sistema (Transact-SQL)](./system-stored-procedures-transact-sql.md). Estos procedimientos están sujetos a cambios y no se garantiza la compatibilidad. Todos los procedimientos de la lista no están disponibles en [!INCLUDE[ssSDW](../../includes/sssdw-md.md)] .

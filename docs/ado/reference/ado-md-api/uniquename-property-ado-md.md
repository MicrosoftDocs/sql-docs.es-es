# <a name="uniquename-property-ado-md"></a>UniqueName (propiedad, ADO MD)
Indica un nombre no ambiguo para el objeto actual.  
  
## <a name="return-values"></a>Valores devueltos  
 Devuelve una **cadena** y es de solo lectura.
